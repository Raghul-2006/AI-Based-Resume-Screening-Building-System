# AI-Based-Resume-Screening-Building-System
The AI-Based Resume Screening and Builder System is an intelligent recruitment platform designed to simplify hiring for recruiters while helping job seekers create ATS-friendly resumes. It addresses the challenge of manually reviewing hundreds of resumes by automating candidate screening and ranking using Artificial Intelligence and NLP.  

### 👨‍💼 HR Recruiter

* Secure HR authentication
* Upload multiple PDF/DOCX resumes
* AI-powered resume screening
* Candidate ranking based on match score
* Skill matching and missing skill detection
* Experience and education analysis
* Resume analytics dashboard
* Candidate profile review
* Application management

### 👤 Job Seeker

* User registration & login
* ATS Resume Builder
* ATS Resume Score Checker
* Browse available jobs
* Apply directly to jobs
* Track application status
* Personal profile management

## 🤖 AI Screening Pipeline

* Resume text extraction (PDF & DOCX)
* NLP-based keyword extraction
* Skill matching against job description
* Experience detection
* Education recognition
* Logistic Regression prediction model
* Composite AI score generation
* Candidate recommendation (Interview / Shortlist / Review)

### Composite Score Formula

* **40%** Keyword Match
* **30%** Machine Learning Prediction
* **20%** Experience Score
* **10%** Education Score
